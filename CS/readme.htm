How to: Create a Wizard with pages defined in XAML

This example demonstrates how to create a simple Wizard (https://documentation.devexpress.com/#WPF/CustomDocument115979) with predefined pages. There are three built-in page (https://documentation.devexpress.com/#WPF/CustomDocument115997) types: - WelcomeWizardPage - WizardPage - CompletionWizardPage These pages have the same functionality, and only their button and content region visibility settings are different. To show/hide page buttons (https://documentation.devexpress.com/#WPF/CustomDocument115998), use the following properties: -ShowBack -ShowCancel -ShowFinish -ShowNext To enable/disable page buttons, use the Allow[Back|Cancel|Finish|Next] properties.
